# Моделирование инфекционного заболевания (SIRVD)

Численная модель эпидемии SIRVD с использованием различных методов:

- Метод Рунге — Кутты порядка 2 (RK2)
- Метод Рунге — Кутты порядка 3 (RK3)
- Метод Рунге — Кутты порядка 4 (RK4)
- Адаптивный метод Рунге — Кутты (RK5(6)) с автоматическим выбором шага `h`

## Работа с моделью

Работа с моделью происходит следующим образом:

1. Необходимо скомпилировать бинарный файл для запуска моделирования.
2. Запустить бинарный файл:
   a. без аргументов — для стандартных параметров
   b. передать аргументы в виде пар «ключ-значение» для тонкой регулировки модели
3. Отобразить графики с помощью модуля Python

## Инструкция

### 0. **Клонирование репозитория**

Выполните клонирование репозитория с помощью `git`:

```bash
git clone https://github.com/lilprovod/sirvd
```

### 1. **Компиляция программы**

Скомпилируйте программу для моделирования (из каталога репозитория):

```bash
gcc -std=c99 src/main.c src/rk.c src/sirvd.c -O2 -Iinclude -lm -o sirvd
```

### 2. **Запуск моделирования**

Для запуска моделирования запустите скомпилированный `.exe` файл:

```bash
.\sirvd
```

Программа принимает аргументы в формате `key=value` для настройки модели. Пример запуска:

```bash
.\sirvd method=rk56 h=0.5 t_end=365 out=result.csv
```

### 3. **Визуализация данных**

В модели предусмотрена визуализация данных с помощью Python. Чтобы отобразить графики, вызовите:

```bash
python -m plot output_name.csv
```

*По умолчанию `output_name.csv` = `output.csv`*
